# 🎬 CGV 김포한강점 이벤트 안내 페이지

[이벤트 페이지 바로가기](https://gmpluckypicky.netlify.app/)

---

## 🚀 프로젝트 개요

CGV 김포한강점에서만 진행되는 이벤트 정보를 손쉽게 제공하기 위해 제작된 웹 페이지입니다.  
매장 방문객들이 대기하는 동안 이벤트 내용을 빠르게 확인할 수 있도록 QR 코드로 접속이 가능하게 구현되었습니다.

---

## 📌 주요 기능

✅ 이벤트 상세 정보 제공  
✅ QR 코드를 통한 간편 접속  
✅ 반응형 웹 지원
